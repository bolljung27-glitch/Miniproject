Point Guard (PG)

ผู้คุมจังหวะเกม รับผิดชอบการจ่ายบอล และควบคุมรูปแบบการบุกของทีม

ตารางฝึกซ้อม Point Guard

วัน เวลา กิจกรรม

อังคาร 16:00 - 18:00 Game Situation / Playmaking
พุธ 16:00 - 18:00 Agility / Conditioning
พฤหัสบดี 16:00 - 18:00 Passing / Defense / Shooting
ศุกร์ 16:00 - 18:00 Scrimmage / Game Tactics

ข้อดี-ข้อเสียของตำแหน่ง Point Guard

ข้อดี ข้อเสีย
ควบคุมเกมและจังหวะของทีมได้ดี ต้องแบกรับความกดดันสูง
มีทักษะการจ่ายบอลและสร้างโอกาสทำคะแนน ต้องมีความฟิตและความเร็วสูงตลอดเกม
ช่วยทีมในการตั้งเกมบุก อาจโดนผู้เล่นสูง/แข็งแรงกดดันได้
